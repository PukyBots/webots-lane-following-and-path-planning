# Webots Lane Following and Path Planning

A robotics simulation project developed in **Webots** demonstrating autonomous **lane following** and **grid-based path planning** for mobile robots.

## Overview

This project combines two fundamental autonomous navigation capabilities:

1. **Lane Following**
   - Detects and follows lane markings.
   - Maintains vehicle alignment within the lane.
   - Simulates autonomous road navigation.

2. **Path Planning**
   - Generates an optimal path between start and goal locations.
   - Avoids obstacles within the environment.
   - Demonstrates autonomous decision-making in structured maps.

## Features

### Lane Following
- Real-time lane tracking
- Autonomous steering control

### Path Planning
- Grid-based navigation

### Simulation Environment
- Fully implemented in Webots

## How to Run

1. Install Webots.
2. Clone this repository:

```bash
git clone https://github.com/yourusername/webots-lane-following-and-path-planning.git
```

3. Open the project world:

```text
worlds/simulation.wbt
```

4. Run the simulation from Webots.
